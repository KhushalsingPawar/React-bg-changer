# React Background Color Changer

A simple React application that changes the background color dynamically when a user clicks on buttons.

## Concepts Used
- React Functional Components
- useState Hook
- Event Handling (onClick)
- JSX
- CSS Styling

## What I Learned
- How to create and use components in React
- How to manage state using useState
- How to handle user actions using events
- How React updates UI without reloading the page

## Tech Stack
- React JS
- Vite
- CSS

## How to Run the Project
1. npm install
2. npm run dev
